# VueJS Demo

This is a simple VueJS project implementing a simple todo list.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files

npm run lint
